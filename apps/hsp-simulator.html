原理と参考文献

1. ハンセン溶解度パラメーター (HSP) とは

物質の溶解性（凝集エネルギー密度）を以下の3つの相互作用に分解してベクトル表現した指標です。

  • δd : 分散項 (Dispersion) - ファンデルワールス力
  • δp : 分極項 (Polarity) - 双極子間力
  • δh : 水素結合項 (Hydrogen bonding) - 水素結合力

2. 溶解性の判定 (HSP距離)

2つの物質（ターゲット1と溶媒2）の間のHSP距離 $Ra$ は以下の式で計算されます。経験的に分散項 $\delta_d$ の寄与が大きいため、係数4が掛けられています。

Ra² = 4(δd₁ - δd₂)² + (δp₁ - δp₂)² + (δh₁ - δh₂)²

判定には、相互作用半径 $R_0$ に対する比率である **RED (Relative Energy Difference)** を用います。

RED = Ra / R₀
  • RED < 1.0 : 良溶媒（溶解・相溶しやすい）
  • RED = 1.0 : 境界
  • RED > 1.0 : 貧溶媒（不溶）

3. 混合溶媒の計算則

混合溶媒のHSP値は、各成分の体積積分率 $\phi$ に基づく加重平均として計算されます（理想混合を仮定）。

δ_mix = Σ (φᵢ × δᵢ)
